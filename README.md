## Overview

DRIVE SAFE ASSIST is a PHP-based web application developed to help drivers quickly find nearby mechanics when their vehicle breaks down or meets with an accident. The platform connects users with registered mechanics, enabling faster roadside assistance and reducing the time spent searching for repair services.

The system consists of three different user roles: **Admin**, **User**, and **Mechanic**, each with its own functionalities.

## Features

### 👤 User
- Register and login
- View nearby mechanics
- Search mechanics
- Send service requests
- View mechanic details
- Manage profile

### 🔧 Mechanic
- Register and login
- Manage mechanic profile
- View customer service requests
- Accept or reject requests
- Update availability

### 👨‍💼 Admin
- Secure admin login
- Manage users
- Manage mechanics
- View and manage service requests
- Monitor overall system

## Project Objective

The primary objective of DRIVE SAFE ASSIST is to provide immediate assistance to drivers by helping them locate nearby mechanics during emergencies, ensuring faster and more convenient vehicle repair services.
